隱私權政策 / Privacy Policy

最近更新日期 / Last Updated: 2026年1月27日

本隱私權政策旨在說明我們的自動化量化分析工具（以下簡稱「本服務」）如何處理透過社交媒體平台 API 獲取的資訊。

This Privacy Policy explains how our Quantitative Automation Tool (the "Service") handles information obtained through social media platform APIs (including but not limited to TikTok, YouTube, Instagram).

1. 我們收集的資訊 / Information We Collect

本服務僅在您明確授權的情況下，透過 OAuth 2.0 驗證獲取必要的權限（如影片上傳權限）。我們不會收集您的帳號密碼、個人聯絡資訊或任何與服務功能無關的數據。

The Service only accesses necessary permissions (such as video upload) via OAuth 2.0 with your explicit consent. We DO NOT collect your passwords, personal contact information, or any data unrelated to the service's functionality.

2. 資訊的使用方式 / How We Use Information

  • 僅用於將本專案生成的量化分析內容上傳至您授權的帳號。 Used solely to upload quantitative analysis content generated by this project to your authorized account.
  • 我們不會將您的數據分享、販售或租賃給任何第三方。 We do not share, sell, or rent your data to any third parties.
  • 不會將您的授權權限用於發佈非本專案生成的內容。 Authorization will not be used to publish content not generated by this project.

3. 數據儲存與安全性 / Data Storage and Security

所有 API 存取權杖（Access Tokens）均加密儲存在受保護的環境中。我們不會建立任何持久性的個人資料庫來追蹤用戶行為，且僅在執行自動化任務時呼叫權杖。

All API Access Tokens are encrypted and stored in a protected environment. We do not maintain persistent databases for user tracking, and tokens are only accessed during automated task execution.

4. 數據保留與刪除 / Data Retention and Deletion

如果您停止使用本服務或撤銷授權，相關的存取權杖將立即從我們的系統中失效並刪除。

If you cease using the Service or revoke authorization, the associated access tokens will be invalidated and deleted from our system immediately.

5. 如何撤銷授權 / How to Revoke Authorization

您可以隨時透過各平台的安全設定頁面撤銷本服務的存取權限（例如：TikTok > 設定與隱私 > 安全與登入 > 管理應用程式授權）。

You can revoke access at any time through each platform's security settings (e.g., TikTok > Settings and Privacy > Security and Login > Manage App Permissions).
