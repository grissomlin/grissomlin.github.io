隱私權政策 (Privacy Policy)

最近更新日期：2026年1月19日

本隱私權政策旨在說明我們的自動化量化分析工具（以下簡稱「本服務」）如何處理透過社交媒體平台（包括但不限於 YouTube, TikTok, Instagram）API 獲取的資訊。

1. 我們收集的資訊

本服務僅在您明確授權的情況下，透過 OAuth 驗證獲取必要的權限（如影片上傳權限）。我們不會收集您的帳號密碼、個人聯絡資訊或任何與自動化發佈無關的數據。

2. 資訊的使用方式

  • 僅用於將本專案生成的量化分析短影音上傳至您授權的帳號。
  • 不會將您的授權權限用於發佈非本專案生成的內容。
  • 我們不會將您的數據分享、販售或租賃給任何第三方。

3. 數據儲存與安全性

所有 API 存取權杖（Access Tokens）均安全地儲存在受保護的環境中，且僅供自動化腳本執行時呼叫。我們不會建立任何持久性的個人資料庫來追蹤用戶行為。

4. 如何撤銷授權

您可以隨時透過各平台的安全設定頁面（如 Google 帳戶安全性設定、TikTok 授權管理）撤銷本服務的存取權限。

5. 聯絡我們

若對本隱私權政策有任何疑問，請透過 GitHub 專案頁面與開發者聯繫。
